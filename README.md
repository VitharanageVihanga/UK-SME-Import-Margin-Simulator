# UK SME Import Margin Simulator

A data-driven Streamlit dashboard that helps UK small and medium enterprises (SMEs) model import profitability, assess financial risk, and forecast margins under varying economic conditions.

Built as a Final Year Project using real trade data from HMRC and ONS.

---

## Features

- **Margin Simulation** — Calculate landed cost and profit margin for any import scenario with adjustable FX rate, shipping cost, tariff, and overhead inputs
- **Sensitivity Analysis** — 11×11 scenario grid showing how margin changes across combinations of FX shock and shipping cost
- **Risk Classification** — Automatic HIGH / MODERATE / LOW risk labelling with confidence band adjustments based on data quality
- **Historical Trend Analysis** — Volatility regime detection, 30/90-day moving averages, and import volume trend analysis using real HMRC data (2018–2024)
- **FX Forecasting** — ARIMA-based 30/90-day exchange rate and margin forecasts with confidence intervals
- **Advanced Risk Metrics** — Value-at-Risk (historical, parametric, Monte Carlo), beta sensitivity analysis, and margin decomposition
- **Stress Testing** — Seven predefined scenarios including Base Case, Mild/Severe FX Shock, Supply Chain Disruption, and Perfect Storm
- **Anomaly Detection** — Isolation Forest model flags unusual margin and cost patterns
- **Market Intelligence** — HS2-to-SITC commodity coverage analysis using merged HMRC + ONS datasets
- **Backtesting** — Walk-forward validation of ARIMA order selection with MAE, RMSE, MAPE, and directional accuracy metrics

## Setup & Installation

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/uk-sme-import-margin-simulator.git
cd uk-sme-import-margin-simulator
```

### 2. Create and activate a virtual environment

```bash
python3 -m venv .venv
source .venv/bin/activate        # macOS / Linux
.venv\Scripts\activate           # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Set up the data (required before first run)

The HMRC monthly import data files (84 files, ~1.9 GB) are not included in this repository due to their size. You must download them separately.

**Step 1 — Download HMRC data**

Download the HMRC Overseas Trade Statistics (OTS) detailed commodity data files for January 2018 through December 2024 from the [HMRC Trade Statistics portal](https://www.uktradeinfo.com/trade-data/ots-custom-table/).

Place the downloaded `.txt` files into:

```
data/raw/hmrc/
```

The files follow the naming convention `BDSimp{YY}{MM}.txt` (e.g. `BDSimp1801.txt` for January 2018).

**Step 2 — Run the ETL pipeline**

Generate the processed and merged datasets:

```bash
python scripts/ons_coverage_by_commodity.py
python scripts/ons_coverage_by_country.py
python scripts/ons_coverage_overall.py
python scripts/classify_ons_coverage_by_commodity.py
```

This populates `data/processed/` and `data/output/`.
